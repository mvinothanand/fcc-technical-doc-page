Complementary Colours

These are colors that are at the opposite sides of a color wheel.

When combined together they become grey. But, when positioned next to each other they complement each other and give a nice visual contrast.

Hue represents the actual color in the light prism but denoted as an angle in the color wheel.

Saturation indicates the amount of gray in the color represented as a percentage.

Lightness indicates the amount of white or black in the color.

repeating-linear-gradient(45deg, yellow 0px, yellow 40px, black 40px, black 80px) --title-color: #FF00FF; --title-color: #FF00FF;

Length Units

Length or size properties of elements can be specified various units. These units can be absolute or relative.

Absolute units are to be avoided as they don't scale based on the device sizes.

Relative units are in relation to another element or the viewport. These units can scale well on different devices.

a[title] // anchor elements with title attribute set a[href="www.example.org"] // elements with href value set to www.example.org a[href$=".org"] // elements with href value ending ".org" a[href*="example"] // elements with href value containing "example" input[type="checkbox"] 1vw=1% of viewport width

Pseudo Elements

::before and ::after pseudo classes require a content property to set mandatorily. Usually this property refers to an image or a text. However, when these pseudo classes are used to create shapes, the property shall be set to empty string.

Using tabindex attribute on a non-interactive element will enable it for this CSS pseudo class.
